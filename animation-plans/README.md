# Listing Copilot motion plan

Motion is reserved for low-frequency state changes. Keyboard-driven writer work stays immediate.

- `001-stabilize-workbench-motion.md`: remove distracting or expensive motion.
- `002-bridge-occasional-state-changes.md`: add small transitions where state would otherwise feel discontinuous.
- `003-deep-purple-clock-wheel.md`: restore the deep-purple product system and give pointer/wheel queue navigation a clock-like depth transition.
- `004-intake-to-workbench-morph.md`: reduce the page to one state stage, move the prepared upload task into a compact top rail, and preserve card identity while switching between list and wheel layouts.
